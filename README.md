# 模式识别与机器学习编程课

本仓库用于上传模式识别与机器学习编程课的素材，包括但不限于 slides，示例代码等素材。

## Usage
### slides
1. 安装 pandoc (ver 3.*)
2. 用 make 编译对应的 slides

```bash
make all [RATIO=16_9(default), 4_3]
make dest/<target_sildes_name>.html # optional command
```

3. 打开对应的 `dest/<target_sildes_name>.html`
